¿como crear una planta de reciclaje de plastico?

para montar una plata de reciclaje debeos tener en cuenta que necessitamos uan mauinaria especifica
para que este proceso se lleve a cabo acontinuacion mecionare una serie de maquinas que necesitamos
para llevar este proceso acabo

MAQUINARIA

  • Molino
  • lavadora de materia si esta sucuio
  • secadora de materia si esta sucio
  • peletizadora

instrucciones

  1. Todo el material tiene que entrar por un proceso de selecion para determinar su tipo y clase
    y no ser contaminado debe realizarse de forma manual.
  2. ya de selecionado en material por colores y tipos se porcede al porceso de molido donde se lleva a
    cabo la desrtruccion del material para que se pueda lavar si esta sucio o si es su defecto esta limpio
    dar el guiente paso
  3. una vez limpio el material o lavado se lleva ala maquina de peletizar para dar destino fina en forma de
    pepas de lentejas o de pasta para que dependioendo la necesidad le puedan dar destino final al matererial
    reciclado.
